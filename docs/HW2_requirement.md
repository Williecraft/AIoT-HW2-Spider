## HW2-1 獲取天氣預報資料 (20%)

### 目的
使用 CWA API 獲取台灣北部、中部、南部、東北部、東部及東南部地區一週的天氣預報資料（必須是 JSON 格式）。

### 方法

### 評分比重

## HW2-3 將氣溫資料儲存到 SQLite3 資料庫 (20%)

### 目的
將氣溫資料儲存到 SQLite3 資料庫，以便後續查詢。

### 方法
- 建立 SQLite3 資料庫，名稱為 `data.db`
- 建立資料表 `TemperatureForecasts`
- 儲存氣溫資料，欄位包含：
  - 主鍵（id）
  - 地區名稱（regionName）
  - 時間（dataDate）
  - 最低氣溫（mint）
  - 最高氣溫（maxt）
- 從資料庫查詢資料驗證：
  - 列出所有地區名稱
  - 列出中部地區的氣溫資料

### 評分比重
- 將氣溫資料儲存到 SQLite3 資料庫（10%）
- 檢查資料是否正確被存入資料庫（5%）
- 程式碼結構與可讀性（5%）
